# Get preferred drug dose range

Get preferred minimum and maximum dose of a specific drug

## Usage

``` r
dose_range(drug, con)
```

## Arguments

- drug:

  Drug name or drug ID

- con:

  A
  [DBIConnection-class](https://dbi.r-dbi.org/reference/DBIConnection-class.html)
  object to a DRP database

## Value

A named list of the minimum and maximum doses
